# Steam Store Games 数据分析

本项目基于 Steam 游戏数据集进行基础统计与可视化分析，输出了可直接查看的分析报告与 Notebook。

## 项目内容

- `Steam platform game data analysis.ipynb`：分析 Notebook
- `steam.csv`：原始数据
- `steam_requirements_data.csv`：系统需求相关数据
- `steam_support_info.csv`：支持信息相关数据
- `index.html`：可直接打开的分析报告（HTML）
- `Steam-store-games数据集分析报告.pdf`：分析报告（PDF）

## 使用方式

1. 直接打开 `index.html` 查看可视化报告。
2. 用 Jupyter 打开 `Steam platform game data analysis.ipynb` 复现分析流程。

## 备注

数据与报告文件均为本地生成的分析产出，后续可根据需求更新。
